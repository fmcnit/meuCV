Javascript é uma linguagem de programação usada para dar mais funcionalidades a uma página Web e seus frameworks estão dominando o mercado de aplicações Web. React, NextJS e Angular são as ferramentas mais usadas em desenvolvimentos web, pois usam tecnologias para criar SPA's.

Spa's ou Single Pages Applications são aplicações web que não recarregam a páigina toda vez que o usuário clica em algum elemento, elas apenas renderizam o contéúdo sem fazer o page reload tornando a experiência do usúario bem mais agradável
